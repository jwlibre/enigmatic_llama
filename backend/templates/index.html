Welcome to the Employees Database API

Instructions

Obtaining records (GET)

localhost:5000/api/get
Parameters may be specified in the URL:
emp_no, first_name, last_name, gender

Deleting records (DELETE)

localhost:5000/api/delete
Parameters may be specified in the URL:
emp_no, first_name, last_name, gender

Adding new records (POST)

localhost:5000/api/add
Data must be specified in JSON format in request body.
Required fields:
emp_no - int (e.g. 10000)
birth_date - string (e.g. 1970-01-01)
first_name - string (e.g. John)
last_name - string (e.g. Smith)
gender - character (M/F)
hire_date - string (e.g. 1970-01-01)

Updating records (PUT)

localhost:5000/api/update
Data must be specified in JSON format in request body.
Records are updated based on emp_no ONLY.
Required fields:
emp_no - int (e.g. 10000) (of record to be updated)
birth_date - string (e.g. 1970-01-01)
first_name - string (e.g. John)
last_name - string (e.g. Smith)
gender - character (M/F)
hire_date - string (e.g. 1970-01-01)
